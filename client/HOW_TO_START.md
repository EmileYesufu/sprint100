## Troubleshooting

**If timeout:**
- Make sure both Mac and iPhone on same WiFi
- Or just use tunnel mode (already enabled)

**If still issues:**
- Try: `npm run start:lan` for local network
- Or use iOS Simulator: press 'i' after `npm start`
