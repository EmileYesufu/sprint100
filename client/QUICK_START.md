### 1. Install Dependencies
```bash
cd client

```


## ðŸ“± Building for Production

### 1. Install EAS CLI
```bash
npm install -g @expo/eas-cli
```

### 2. Login to Expo
```bash

```

### 3. Configure EAS
```bash

```

### 4. Build for iOS
```bash

npm run build:development

# Preview build
npm run build:preview

# Production build
npm run build:production
```

### 5. Submit to App Store
```bash
# Submit to TestFlight
npm run submit:ios

# Submit to Play Store
npm run submit:android
```

## ðŸš¨ Troubleshooting

### Common Issues
1. **Metro bundler not starting**: Run `npx expo start --clear`
2. **Build failing**: Check `eas build:list` for error details
3. **Credentials issues**: Run `eas credentials:configure`
4. **API connection**: Verify EXPO_PUBLIC_API_URL in .env

### Debug Commands
```bash
# Check build status
eas build:list

# View build logs
eas build:view [BUILD_ID]

# Check credentials
eas credentials:list

# Clear Metro cache
npx expo start --clear
```
